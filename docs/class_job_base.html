详细描述

JudgeJob 与 UpdateJob 的基类， 存有 Job的基本信息，并规定了所有 Job 必须遵循的函数接口。

注解
本类没有具体意义，仅是所有 Job 的共有特征，因此为抽象类，不可实例化。
