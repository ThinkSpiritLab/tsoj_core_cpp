◆ join() [2/3]

pid_type process::join ( int *  stat_loc,
int  options 
)
inlinenoexcept

Wait for a child matching PID to die.

参数
status_locThe location where the process status will be put.
返回
If the WNOHANG bit is set in OPTIONS, and that child is not already dead, return (pid_t) 0. If successful, return PID and store the dead child's status in STAT_LOC. Return (pid_t) -1 for errors. If the WUNTRACED bit is set in OPTIONS, return status for stopped children; otherwise don't.
